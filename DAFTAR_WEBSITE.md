# Daftar Website Instansi Pemerintah (KASULAMPUA)

Berikut adalah daftar website instansi pemerintah untuk wilayah **KASULAMPUA** (Kalimantan, Sulawesi, Maluku, Papua) serta instansi pusat terkait:

*   **BPS Republik Indonesia:** [bps.go.id](https://www.bps.go.id/)

*   **Sulawesi:** [Sulut](https://sulut.bps.go.id/), [Sulteng](https://sulteng.bps.go.id/), [Sulsel](https://sulsel.bps.go.id/), [Sultra](https://sultra.bps.go.id/), [Gorontalo](https://gorontalo.bps.go.id/), [Sulbar](https://sulbar.bps.go.id/)
*   **Maluku:** [Maluku](https://maluku.bps.go.id/), [Malut](https://malut.bps.go.id/)
*   **Papua:** [Papua](https://papua.bps.go.id/), [Papua Barat](https://papuabarat.bps.go.id/)
